# AutoCircle
Perfect circle drawing tool for https://vole.wtf/perfect-circle/
Gets 99.9% consistently

1. pip install pyautogui
2. pip install pynput
3. pip install selenium
4. python script.py

*Note that this will only work on 1920x1080 screens*
